# LeetCode God Mode

LeetCode God Mode is a powerful Chrome Extension designed to automate the process of solving LeetCode problems. It helps users maintain their daily streak and practice coding by automatically fetching and injecting solutions.

## Features

-   **Automated Solving**: Automatically finds and solves LeetCode problems.
-   **Daily Targets**: Set a daily limit for the number of problems to solve.
-   **Stop Button**: Immediately halt the automation process with a single click.
-   **Solution Sources**: Fetches optimal solutions from LeetCode Community and trusted GitHub repositories (WalkCCC, Kamyu104).
-   **C++ Enforcement**: Prioritizes and validates C++ solutions.
-   **Auto-Login**: Handles login flows if credentials are saved.
-   **Real-time Logs**: View detailed logs of the extension's activity in the popup.

## Setup & Installation

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/noeankx/Leetcode_solver.git
    cd Leetcode_solver
    ```

2.  **Load into Chrome**:
    -   Open Chrome and navigate to `chrome://extensions/`.
    -   Enable **Developer mode** (toggle in the top right).
    -   Click **Load unpacked**.
    -   Select the `Leetcode_solver` directory.

3.  **Configure**:
    -   Click the extension icon.
    -   Enter your LeetCode credentials (optional, for auto-login).
    -   Set your **Daily Target**.
    -   Click **Save**.

## Usage

-   **Start**: Click the **Start Solving** button in the popup. The extension will open LeetCode and begin the automation loop.
-   **Stop**: Click the **Stop** button to immediately pause the bot.
-   **Logs**: Monitor progress in the "Status" console within the popup.
